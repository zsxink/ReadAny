ReadAny 是一款 AI 驱动的电子书阅读器，支持 EPUB、PDF、MOBI、AZW、AZW3、FB2、CBZ 等格式。提供智能对话、语义搜索、标注笔记与知识管理功能，所有数据本地存储。

## 快速上手

1. 从[下载页面](https://codedogqby.github.io/ReadAny/zh/#download)**下载** ReadAny
2. 在你的平台上**安装**应用
3. **导入**电子书 — 拖拽文件到书架或点击导入按钮
4. **开始阅读** — 点击任意书籍即可打开

## 核心功能

- **多格式支持** — EPUB、PDF、MOBI、AZW、AZW3、FB2、FBZ、CBZ
- **AI 对话** — 基于书籍内容的智能问答
- **语义搜索** — 按语义查找段落，而不仅是关键词
- **智能标注** — 5 种颜色高亮，Markdown 笔记
- **语音朗读** — 内置 TTS，支持多种语音
- **翻译** — 段落级即时翻译
- **阅读统计** — 追踪阅读时间和习惯
- **WebDAV 同步** — 跨设备同步进度和标注
- **导出** — 导出笔记到 Obsidian、Notion 或 Markdown
