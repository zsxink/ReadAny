## macOS

### Apple Silicon (M1/M2/M3/M4)

1. 从[发布页面](https://github.com/codedogQBY/ReadAny/releases)下载 Apple Silicon 版本的 `.dmg` 文件
2. 打开 `.dmg` 文件，将 ReadAny 拖到「应用程序」文件夹
3. 首次启动如果 macOS 提示安全警告，前往**系统设置 → 隐私与安全性**，点击「仍要打开」

### Intel

1. 从[发布页面](https://github.com/codedogQBY/ReadAny/releases)下载 Intel 版本的 `.dmg` 文件
2. 安装步骤同上

## Windows

1. 从[发布页面](https://github.com/codedogQBY/ReadAny/releases)下载 `.msi` 或 `.exe` 安装包
2. 运行安装程序，按提示完成安装
3. 安装完成后可在开始菜单找到 ReadAny

## Linux

1. 从[发布页面](https://github.com/codedogQBY/ReadAny/releases)下载 `.AppImage` 或 `.deb` 安装包
2. AppImage：执行 `chmod +x ReadAny.AppImage` 后运行
3. deb：执行 `sudo dpkg -i ReadAny.deb` 安装

## 更新

ReadAny 内置自动更新功能。有新版本时，**设置 → 关于** 页面会显示更新提示。点击「下载更新」即可在线更新。

也可以手动检查更新：**设置 → 关于 → 检查更新**。
