## 支持的格式

| 格式 | 扩展名 | 说明 |
|---|---|---|
| EPUB | `.epub` | 支持 EPUB 2 和 EPUB 3 |
| PDF | `.pdf` | 文字型和扫描型 |
| MOBI | `.mobi` | Amazon Kindle 格式 |
| AZW | `.azw` | Amazon Kindle 格式 |
| AZW3 | `.azw3` | Amazon KF8 格式 |
| FB2 | `.fb2` | FictionBook 格式 |
| FBZ | `.fbz` | 压缩的 FictionBook |
| CBZ | `.cbz` | 漫画压缩包 |

## 导入方式

### 拖拽导入

直接从文件管理器拖拽电子书文件到 ReadAny 书架窗口。书籍会被自动导入，元数据（标题、作者、封面）会自动提取。

### 导入按钮

1. 点击书架工具栏的 **「+」** 按钮
2. 在文件选择框中选择一个或多个文件
3. 书籍将被添加到书架

### 文件关联

安装后，双击任意支持的电子书文件即可直接在 ReadAny 中打开。

## 书架管理

- **搜索** — 使用搜索栏按标题或作者查找书籍
- **排序** — 按标题、作者、最近阅读或添加时间排序
- **标签** — 添加标签来分类整理书籍
- **删除** — 右键点击书籍，选择「移除」从书架删除
