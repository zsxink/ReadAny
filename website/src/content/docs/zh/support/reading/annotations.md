## 高亮

1. 在阅读器中选中文本
2. 从弹出的工具栏中选择高亮颜色
3. 高亮会自动保存

可用颜色：**黄色**、**绿色**、**蓝色**、**粉色**、**紫色**

## 添加笔记

1. 选中文本后点击工具栏中的**笔记**图标
2. 使用 Markdown 语法编写笔记
3. 点击编辑器外部即可保存

笔记支持完整的 Markdown 语法，包括标题、列表、代码块和链接。

## 管理标注

打开侧边栏的**标注面板**可以：
- 查看所有高亮和笔记（按章节分组）
- 点击任意标注跳转到对应位置
- 编辑或删除已有笔记
- 按颜色筛选标注

## 导出标注

前往**标注面板 → 导出**，选择导出格式：

| 格式 | 说明 |
|---|---|
| Markdown | 纯 Markdown，按章节分组 |
| JSON | 结构化数据格式 |
| Obsidian | 带 frontmatter、`[[双向链接]]` 和高亮块 |
| Notion | 适合粘贴到 Notion 页面 |
