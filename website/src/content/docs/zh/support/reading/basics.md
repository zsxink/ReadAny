## 打开书籍

点击书架中的任意书籍即可打开阅读器。ReadAny 会自动记住并恢复你上次的阅读位置。

## 导航

| 操作 | 快捷键 |
|---|---|
| 下一页 | `→` 或 `空格` |
| 上一页 | `←` 或 `Shift+空格` |
| 目录 | `T` |
| 书内搜索 | `Ctrl/Cmd + F` |
| 全屏切换 | `F11` 或 `Ctrl/Cmd + Enter` |

其他导航方式：
- **目录** — 点击工具栏的目录图标，跳转到任意章节
- **进度条** — 点击或拖拽底部进度条，跳转到指定位置
- **页面滑块** — 使用滑块快速浏览

## 文本选择

在阅读器中选中文本，弹出操作工具栏：
- **高亮** — 从 5 种颜色中选择
- **笔记** — 添加 Markdown 笔记
- **复制** — 复制到剪贴板
- **翻译** — 翻译选中文本
- **问 AI** — 就选中文本向 AI 提问

## 侧边栏面板

- **目录** — 章节导航
- **标注** — 查看和管理所有高亮和笔记
- **搜索** — 书内全文搜索
- **AI 对话** — 与 AI 讨论书籍内容
- **思维导图** — 章节结构可视化
