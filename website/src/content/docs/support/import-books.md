## Supported Formats

ReadAny supports the following e-book formats:

| Format | Extensions | Notes |
|---|---|---|
| EPUB | `.epub` | EPUB 2 and EPUB 3 |
| PDF | `.pdf` | Reflowable text and scanned documents |
| MOBI | `.mobi` | Amazon Kindle format |
| AZW | `.azw` | Amazon Kindle format |
| AZW3 | `.azw3` | Amazon KF8 format |
| FB2 | `.fb2` | FictionBook format |
| FBZ | `.fbz` | Compressed FictionBook |
| CBZ | `.cbz` | Comic book archive |

## Import Methods

### Drag & Drop

Simply drag one or more e-book files from your file manager into the ReadAny library window. The books will be automatically imported and their metadata (title, author, cover) extracted.

### Import Button

1. Click the **"+"** button in the library toolbar
2. Select one or more files from the file dialog
3. The books will be added to your library

### File Association

After installation, you can double-click any supported e-book file to open it directly in ReadAny.

## Library Management

- **Search** — Use the search bar to find books by title or author
- **Sort** — Sort your library by title, author, recently read, or date added
- **Tags** — Add tags to organize your books into collections
- **Delete** — Right-click a book and select "Remove" to delete it from your library
