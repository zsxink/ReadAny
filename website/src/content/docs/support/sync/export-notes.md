## Export Formats

| Format | Best For |
|---|---|
| **Markdown** | General use, plain text notes |
| **JSON** | Programmatic access, data processing |
| **Obsidian** | Obsidian vault integration with wiki-links |
| **Notion** | Pasting into Notion pages |

## How to Export

1. Open a book in the reader
2. Open the **Annotations** panel in the sidebar
3. Click the **Export** button
4. Select your desired format
5. The file will be saved or copied to clipboard

## Export Content

Each export includes:
- Book metadata (title, author)
- All highlights grouped by chapter
- Your notes in Markdown format
- Color labels for each highlight
- Chapter and position references

## Obsidian Integration

When exporting in Obsidian format, the output includes:
- YAML frontmatter with book metadata
- `[[wiki-links]]` for cross-referencing
- Callout blocks for different highlight colors
- Compatible with Obsidian's folder structure
