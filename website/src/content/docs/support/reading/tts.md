## Using TTS

1. Open a book in the reader
2. Click the **speaker icon** in the toolbar or press the TTS shortcut
3. ReadAny will start reading from your current position

## Controls

- **Play/Pause** — Toggle TTS playback
- **Stop** — Stop and reset TTS
- **Speed** — Adjust reading speed (0.5x to 3x)
- **Skip** — Jump forward or backward by sentence

## TTS Settings

Configure TTS in **Settings → TTS**:

- **Voice** — Select from available system voices
- **Speed** — Default reading speed
- **Auto-advance** — Automatically turn pages and continue reading

## Platform Notes

| Platform | TTS Engine |
|---|---|
| macOS | System Speech Synthesis (high quality) |
| Windows | Microsoft Speech API |
| Linux | System speech-dispatcher |
| iOS/Android | Native platform TTS |
