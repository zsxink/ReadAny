## macOS

### Apple Silicon (M1/M2/M3/M4)

1. Download the `.dmg` file for Apple Silicon from the [releases page](https://github.com/codedogQBY/ReadAny/releases)
2. Open the `.dmg` file and drag ReadAny to your Applications folder
3. On first launch, if macOS shows a security warning, go to **System Settings → Privacy & Security** and click "Open Anyway"

### Intel

1. Download the Intel `.dmg` file from the [releases page](https://github.com/codedogQBY/ReadAny/releases)
2. Follow the same steps as above

## Windows

1. Download the `.msi` or `.exe` installer from the [releases page](https://github.com/codedogQBY/ReadAny/releases)
2. Run the installer and follow the on-screen instructions
3. ReadAny will be available in your Start menu

## Linux

1. Download the `.AppImage` or `.deb` package from the [releases page](https://github.com/codedogQBY/ReadAny/releases)
2. For AppImage: make it executable with `chmod +x ReadAny.AppImage` and run it
3. For .deb: install with `sudo dpkg -i ReadAny.deb`

## Updating

ReadAny includes a built-in auto-updater. When a new version is available, you'll see a notification in **Settings → About**. Click "Download Update" to update in place.

You can also manually check for updates via **Settings → About → Check for Updates**.
